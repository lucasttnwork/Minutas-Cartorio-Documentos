================================================================================
COMPONENTES (src/components/streaming/)
================================================================================

StreamingText.tsx + StreamingText.css
- Typing animation básica
- Props: text, speed, isComplete, showCursor, batchSize
- Cursor piscando integrado
- Performance otimizada

ThinkingIndicator.tsx + ThinkingIndicator.css
- Indicador de reasoning/thinking
- Props: content, isStreaming, duration, autoCollapse
- Expandable/collapsible
- Shimmer effect durante streaming
- Mostra "Pensou por Xs"

TypingIndicator.tsx + TypingIndicator.css
- Dots animados de carregamento
- Props: isVisible, size (small/medium/large), label
- Bounce animation
- Acessível para screen readers

StreamingChatMessage.tsx + StreamingChatMessage.css
- Componente completo de mensagem
- Props: id, role, content, thinking, isStreaming
- Integra tudo (thinking + streaming + status)
- Auto-scroll
- Estilo light/dark com gradientes

index.ts
- Exports convenientes para todos os componentes
